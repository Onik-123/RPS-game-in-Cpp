# RPS-game-in-Cpp
A simple Rock, Paper, Scissors game implemented in C++. Play against the computer, make your choice, and see if you can beat the odds! Features include input validation, score tracking, and replay options.

Rock Paper Scissors Game
A simple C++ console game where you choose Rock (0), Paper (1), or Scissors (2) against a random computer choice.
How to Play

Run, play and have FUN! :)

Files

choice.hpp: Defines choices.

player.hpp: Handles input.

computer.hpp: Generates computer choice.

engine.hpp: Manages game logic.

main.cpp: Runs the game.
